# VGICs-pred
VGICs-pred: A protein language model-based tool for voltage-gated ion channel (VGIC) sequence annotation and sequence determinant discovery. It integrates ProtT5 embeddings with a multi-layer perceptron, enabling high-accuracy VGIC classification from amino acid sequence alone.
This repo includes full inference/training code, pre-trained models, data augmentation, feature analysis pipelines and datasets for high-throughput ion channel annotation and mechanistic research.
